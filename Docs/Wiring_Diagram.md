# 接线说明

## 自动售货机 PLC 控制系统

---

## 电源接线

```
AC 220V (±10%) 50Hz
    │
    ├──→ PLC 电源端子 (L+, M) ──→ DC 24V 输出
    │
    └──→ 开关电源 (AC→DC 24V, ≥2A)
              │
              ├──→ PLC 数字量输入公共端
              ├──→ PLC 数字量输出公共端
              └──→ 各执行机构（电机）
```

---

## PLC 端子接线图 (CPU 1214C DC/DC/DC)

### 输入端子 (I)

```
1M ──→ DC 24V (公共端)

I0.0 ──→ [BTN_1YUAN] ──→ 1M    (1元投币按钮)
I0.1 ──→ [BTN_2YUAN] ──→ 1M    (2元投币按钮)
I0.2 ──→ [BTN_5YUAN] ──→ 1M    (5元投币按钮)
I0.3 ──→ [BTN_10YUAN]──→ 1M    (10元投币按钮)
I0.4 ──→ [BTN_COLA]  ──→ 1M    (可乐选择)
I0.5 ──→ [BTN_JUICE] ──→ 1M    (果汁选择)
I0.6 ──→ [BTN_MILKTEA]──→ 1M   (奶茶选择)
I0.7 ──→ [BTN_WATER] ──→ 1M    (矿泉水选择)
I1.0 ──→ [BTN_COFFEE]──→ 1M    (咖啡选择)
I1.1 ──→ [BTN_TEA]   ──→ 1M    (茶选择)
I1.2 ──→ [BTN_CANCEL]──→ 1M    (取消/退款)
I1.3 ──→ [BTN_COMBO] ──→ 1M    (套餐购买)
I1.4 ──→ [BTN_CONFIRM]──→1M    (确认出货)
```

### 输出端子 (Q)

```
1L+ ──→ DC 24V (输出公共电源)

Q0.0 ──→ [继电器] ──→ [MOTOR_COLA]    (可乐出货电机)
Q0.1 ──→ [继电器] ──→ [MOTOR_JUICE]   (果汁出货电机)
Q0.2 ──→ [继电器] ──→ [MOTOR_MILKTEA] (奶茶出货电机)
Q0.3 ──→ [继电器] ──→ [MOTOR_WATER]   (矿泉水出货电机)
Q0.4 ──→ [继电器] ──→ [MOTOR_COFFEE]  (咖啡出货电机)
Q0.5 ──→ [继电器] ──→ [MOTOR_TEA]     (茶出货电机)
Q0.6 ──→ [继电器] ──→ [MOTOR_CHANGE]  (找零电机)
Q0.7 ──→ [LED]    ──→ [LAMP_READY]    (就绪指示灯)
```

## 执行机构接线说明

### 出货电机（模拟）
```
DC 24V (+) ──→ 继电器常开触点 ──→ 电机 (+)
DC 24V (-) ──→ 电机 (-)

PLC Q0.x ──→ 继电器线圈 (+)
继电器线圈 (-) ──→ DC 24V (-)
```

### 找零电机（模拟）
```
接线方式同上，由 Q0.6 驱动
```

### 指示灯
```
PLC Q0.7 ──→ 限流电阻 (1kΩ) ──→ LED (+)
LED (-) ──→ DC 24V (-)
```

## 注意事项

1. **所有按钮建议加 RC 吸收电路**（0.1μF + 100Ω），防止触点抖动
2. **继电器线圈加续流二极管**（1N4007），保护 PLC 输出
3. **电机建议加过压保护**（压敏电阻 TVS）
4. **接线完成后先空载测试**，确认逻辑正确再连接实际负载
5. **PLC 输出端最大电流 0.5A/点**，大功率电机必须通过继电器驱动
6. **DC 24V 电源容量** ≥ 2A，建议选用 3A 开关电源留余量
