# HMI 画面设计指南

## TIA Portal V18 WinCC 画面配置说明

本项目需要在 HMI（KTP700 Basic / WinCC 仿真）中创建以下画面：

---

## 📱 画面列表

### 1. 主画面 (MainScreen)
**功能**：售货机主操作界面

**布局设计**：
```
┌──────────────────────────────────────────┐
│          ★ 自动售货机 ★                   │
│                                          │
│   ┌──────────────────────────────────┐   │
│   │  欢迎光临，请投币                │   │
│   │  当前金额: 0 元                  │   │
│   └──────────────────────────────────┘   │
│                                          │
│   ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐      │
│   │ 🥤  │ │ 🧃  │ │ 🧋  │ │ 💧  │      │
│   │可乐 │ │果汁 │ │奶茶 │ │矿泉水│      │
│   │ 3元 │ │ 4元 │ │ 5元 │ │ 2元 │      │
│   │ [5] │ │ [5] │ │ [5] │ │ [5] │      │
│   └─────┘ └─────┘ └─────┘ └─────┘      │
│   ┌─────┐ ┌─────┐                       │
│   │ ☕  │ │ 🍵  │   ┌───────────┐       │
│   │咖啡 │ │ 茶  │   │  套餐区   │       │
│   │ 6元 │ │ 3元 │   │ 可乐套餐  │       │
│   │ [5] │ │ [5] │   │  7元     │       │
│   └─────┘ └─────┘   └───────────┘       │
│                                          │
│   ┌────┐┌────┐┌────┐┌────┐┌────┐       │
│   │ 1元││ 2元││ 5元││10元││取消│       │
│   └────┘└────┘└────┘└────┘└────┘       │
│                                          │
│   [管理入口]          [确认购买]         │
└──────────────────────────────────────────┘
```

**所需控件**：
- **文本域**：显示 `DisplayText`（主显示文字）和 `WelcomeText`
- **IO域**：显示 `TotalAmount`（当前金额），格式 `# 元`
- **按钮**（各6个商品按钮）：
  - 可乐按钮 → 脚本设置 `M10.0 = 1`（保持200ms后复位）
  - 也可直接关联 PLC 的 I0.4-I1.1
- **按钮**：投币按钮（1元/2元/5元/10元）
- **按钮**：取消（关联 M10.4）、确认（关联 I1.4）
- **按钮**：套餐购买（关联 I1.3）
- **按钮**：管理入口 → 切换到 AdminLogin 画面
- **图形**：商品图标（饮料图片）
- **状态指示**：库存显示（`Stock_Cola` 等）

### 2. 商品详情区
每个商品位置需要显示：
- **商品图片**（静态图标）
- **商品名称**（文本）
- **商品价格**（IO域绑定 `Price_xxx`）
- **库存数量**（IO域绑定 `Stock_xxx`）
- **售罄覆盖层**：当 `SoldOut_xxx = TRUE` 时显示红色"售罄"文字
- **低库存闪烁**：当 `LowStock_xxx = TRUE` 时图片闪烁动画

### 3. 套餐选择区
- 显示套餐名称和优惠价
- 套餐可用性指示
- "套餐购买"按钮

### 4. 管理员登录画面 (AdminLogin)
**功能**：密码验证

**控件**：
- 密码输入框（IO域，绑定 `AdminPassword`）
- "登录"按钮
- "返回"按钮

### 5. 管理界面 (AdminScreen)
**功能**：库存管理、销售统计、价格设置

**布局**：
```
┌──────────────────────────────────────┐
│         管理后台                      │
│                                      │
│  [库存管理] [销售统计] [价格设置]     │
│                                      │
│  ┌─────── 库存管理 ───────┐         │
│  │ 商品    库存   状态     │         │
│  │ 可乐    [ 5]   正常    │         │
│  │ 果汁    [ 5]   正常    │         │
│  │ 奶茶    [ 2]   ⚠ 低   │         │
│  │ 矿泉水  [ 0]   ✘ 售罄  │         │
│  │ 咖啡    [ 5]   正常    │         │
│  │ 茶      [ 5]   正常    │         │
│  │                         │         │
│  │  [一键补货]  [补货提醒] │         │
│  └─────────────────────────┘         │
│                                      │
│  [返回]   [退出登录]                  │
└──────────────────────────────────────┘
```

**控件**：
- 表格/IO域：显示每种商品的库存和状态
- "一键补货"按钮：触发 `HMI_BtnRestock`
- 库存输入框：可直接修改库存值

### 6. 销售统计画面 (SalesScreen)
**功能**：显示销售数据

**控件**：
- 总交易笔数（绑定 `TotalTransactions`）
- 总销售额（绑定 `TotalRevenue`）
- 各商品销售量（绑定 `SoldQty_xxx`）
- 饼图/柱状图（可选）

### 7. 价格设置画面 (PriceScreen)
**功能**：管理员修改商品价格

**控件**：
- 每种商品的价格输入框（绑定 `Price_xxx`，可读写）
- "保存"按钮
- "取消"按钮

### 8. 报警画面 (AlarmScreen)
**功能**：显示当前报警

**控件**：
- 报警指示灯（绑定 `AlarmActive`，红色闪烁）
- 报警文字（绑定 `AlarmText`）
- 报警列表控件

## ⚡ 动画配置

1. **低库存闪烁**：商品图片，周期 500ms，条件 `LowStock_xxx = TRUE`
2. **售罄覆盖**：红色半透明遮罩，条件 `SoldOut_xxx = TRUE`
3. **出货动画**：出货口箭头动画，条件 `SysState = 3`
4. **找零动画**：找零口闪烁，条件 `SysState = 5`
5. **欢迎文字**：文字滚动效果

---

## 📝 创建步骤

1. 在 TIA Portal V18 中右键 HMI 项目 → "Add new screen"
2. 按上述列表创建各个画面
3. 导入 `HMI_Tags.sdf` 变量表
4. 逐一添加控件并绑定变量
5. 配置画面切换关系
6. 设置报警和动画
7. 下载测试

> **提示**：如果没有实体HMI，可以使用 TIA Portal 内置的 WinCC 仿真器运行测试。
